## Anaconda

1. Download do anaconda
2. <https://www.anaconda.com/products/individual>
3. <https://docs.anaconda.com/anaconda/install/linux/>
4. $ sudo chmod +x Anaconda3-2020.02-Linux-x86_64.sh
5. $ ./Anaconda3-2020.02-Linux-x86_64.sh
6. fechar o terminal e abrir de novo
7. $ source ~/.bashrc
8. $ /home/mt/anaconda3/bin/activate
9. $ conda list
10. $ anaconda-navigator
11. se nao funcionar
12. $ conda install -c anaconda anaconda-navigator
13. $ cd anaconda3/bin
14. $ ./anaconda-navigator
15. terminar
16. $ conda config --set auto_activate_base False
17. $ conda deactivate
